Wine Reviews

1. 数据可视化和摘要

1.1 数据集

类型 列名 描述
A country The country that the wine is from
A description The country that the wine is from
A designation The vineyard within the winery where the grapes that made the wine are from
# points The number of points WineEnthusiast rated the wine on a scale of 1-100 (though they say they only post reviews for wines that score >=80)
# price The cost for a bottle of the wine
A province The province or state that the wine is from
A region_1 The wine growing area in a province or state (ie Napa)
A region_2 Sometimes there are more specific regions specified within a wine growing area (ie Rutherford inside the Napa Valley), but this value can sometimes be blank
A taster_name 品酒师姓名
A taster_twitter_handle
A title The title of the wine review, which often contains the vintage if you’re interested in extracting that feature
A variety The type of grapes used to make the wine (ie Pinot Noir)
A winery The winery that made the wine

Consumer & Visitor Insights For Neighborhoods

1. 数据可视化和摘要

1.1 数据集

类型 列名 描述
census_block_group String Census Block Group独特的12位FIPS代码。请注意，一些CBG具有前导零
date_range_start： Long 测量周期的开始时间为UTC秒的时间戳。
date_range_end： Long 测量周期的结束时间作为UTC秒的时间戳。
raw_visit_count： Integer 在日期范围内，我们的小组对此CBG的访问次数。
raw_visitor_count： Integer 在日期范围内，我们的小组看到此POI的唯一身份访问者数量。
visitor_home_cbgs： {String:Int} 某个社区观察到的访问人数
visitor_work_cbgs： {String:Int} 访问某个CBG的
distance_from_home： Integer 访客到家的距离
related_same_day_brand： Array 此CBG的参观者在访问CBG的同一天访问的品牌，其中客户重叠与SafeGraph全国平均值相比至少相差5％。
related_same_month_brand： Array 此CBG的访客在访问CBG的同一个月访问过的品牌，其中客户重叠与SafeGraph全国平均值相差至少5％。
top_brands： Array ，在此期间，CBG中访问过的顶级品牌列表。仅限十大品牌。
popularity_by_hour： Array 在当地时间的日期范围内，将当天的小时数与每小时的访问次数映射。
popularity_by_day： Array 在日期范围内，每周的日期映射到每天（当地时间）的访问次数。
